# 网球记分工具 🎾

手机端单文件网页，用于网球比赛实时计分：**语音 / 点按记分**，自动**语音播报**比分，并支持 **Apple Watch** 远程点按记分。零依赖、纯静态。

## 功能

- **赛制**：1 / 3 / 5 / 7 局几胜可选
- **姓名**：手动输入双方姓名（1号 / 2号）
- **记分方式**（三选一，可混用）：
  - 屏幕大按钮点按
  - 语音说「一号得分」「二号得分」「撤销得分」「报比分」——带**模糊匹配**，容忍「壹号 / 幺号 / 七号」等语音误识别
  - Apple Watch 快捷指令点按（经 ntfy.sh 频道转发）
- **标准网球计分**：15 / 30 / 40 / 平分 / 占先，局点 / 赛点提示
- 每次得分**自动语音播报**当前轮次与比分，并直接念选手姓名
- 撤销、重报比分、屏幕常亮、各局回顾

## 使用

用手机浏览器（Chrome / Safari）打开 `index.html` 即可，需允许使用麦克风、建议开外放并调大音量。

## Apple Watch 记分（可选）

原理：页面通过 EventSource 订阅 ntfy.sh 的一个随机频道；Apple Watch 上的「快捷指令」用「获取 URL 内容」(GET) 向该频道 publish 一条消息（如 `一号得分`），页面收到后即记分并播报。

一次性配置：

1. 在页面里分别复制「一号得分 / 二号得分 / 撤销得分」三条链接
2. iPhone「快捷指令」App → 新建快捷指令 → 加「获取 URL 内容」操作 → 粘贴链接 → 打开「在 Apple Watch 上显示」
3. **倒序创建**（先撤销得分、再二号得分、最后一号得分），这样手表上从上到下正好是 一号 / 二号 / 撤销得分

> ⚠️ ntfy.sh 为公共消息服务，需手机和手表都联网；频道名随机生成，别人猜不到就收不到比分。

## 技术

单文件 `index.html`，无构建、无依赖。用到 Web Speech API（语音识别 + 合成）、Screen Wake Lock API、EventSource。
